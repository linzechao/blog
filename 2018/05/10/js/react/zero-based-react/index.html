1. Yarn

我使用 Yarn 包管理器，体验上确实比 npm/cnpm 安装与更新更快，更便捷，维护起来也比较轻松。与 npm（由 Nodejs 安装自带）一样需要安装 exe 程序，下载地址。常用命令也就那几个，但要从 npm 的习惯中抽离出来是需要一段时间来过渡。

  • Mock.js 官网
