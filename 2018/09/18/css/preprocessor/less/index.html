Less 是一门 CSS 预处理器

2018-09-18
less

Less 是一门 CSS 预处理语言，它扩展了 CSS 语言，增加了变量、Mixin、函数等特性，使 CSS 更易维护和扩展。如果你只是开发一个小项目，根本就用不上什么高大上的构建工具，也没必要去安装 Sass 需要的 Ruby 环境，其实 Less 才是你的首选！
