CSSSASS

最火的样式预处理器 SASS

2018-09-15
sass

世界上最成熟、最稳定、最强大的专业级 CSS 扩展语言！Sass 与 Less，只是语法上的差别，功能上该有的都有，你有我也有！但 Sass 提供了类似 Python 语言代码格式与 css 通用格式两种格式，这个未免也贴心了吧！而最大的差别在于，有基于 Sass 开发的 Compass，Less 则需要自己手动写类似 Compass 的组件样式。但经过前端技术更新迭代之后，各种自动化工具出来后，Compass 也慢慢落下帷幕，慢慢的就只剩下习惯用哪个就用哪个的。怎么好看怎么办～

安装

如果不是使用自动化构建工具，如 Grunt/Glup/Webpack 等，那么就需要命令行方式，那就需要安装 Ruby 环境；或者你只是开发一个很简单、几个页面的小项目，那么还是建议使用编译工具 - Koala（考拉）。具体详情可以参考详情。
